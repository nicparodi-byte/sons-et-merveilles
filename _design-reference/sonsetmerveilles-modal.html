États du modal — référence pour développeur Next.js

Avec page dédiée

Le bouton "En savoir plus sur le projet" apparaît. Dans Sanity, le champ slug est renseigné. Dans Next.js, le composant vérifie projet.slug avant de rendre le CTA.

Sans page dédiée

Le bouton "En savoir plus" est absent. Seul le CTA "Parlons de votre projet" est affiché. Le champ slug est vide dans Sanity.

Avec vidéo

Le bloc vidéo s'affiche quand projet.videoEmbedUrl est renseigné dans Sanity. L'iframe YouTube ou Vimeo remplace le placeholder.

Sans vidéo

Le bloc vidéo est masqué. Le modal reste équilibré avec l'audio Spotify et la description. Le champ videoEmbedUrl est vide dans Sanity.

Sanity Schema — Champs requis pour le modal

{
  title:           string        // "Paroles de patients..."
  client:          string        // "Abbvie × LauMa"
  format:          string        // "Témoignage"
  sector:          string        // "Santé"
  description:     text          // Court paragraphe (teaser)
  artwork:         image         // Visuel principal
  spotifyEmbedUrl: url           // URL épisode Spotify
  videoEmbedUrl:   url | null    // Optionnel — YouTube/Vimeo
  slug:            slug | null   // Optionnel — page dédiée
}
